Program Overview

This One-year Graduate Certificate program prepares students to work as multimedia designers and producers. The knowledge and experience gained are now standard in a wide variety of fields.

Graduates may find employment in dynamic media development, web design, server-side development, photography, videography, motion graphics, project management, UX design, graphic design and layout, online education. Other opportunities involve start-up businesses and a wide variety of alternate professional fields.

Courses

Level 1

Web Design

Web Design
  • Students learn HTML and CSS to create structure and organize web-based media.
  • Students learn and practice industry-standard coding techniques.
  • Students learn how to use industry standard software packages, used for layout, programming and developing web pages and sites.

Graphic Applications

Graphic Applications
  • Students learn how to use Adobe Photoshop and Illustrator, the current industry-standard in graphic design and editing software.
  • Students learn the fundamentals of digital design and layout.
  • Students learn the creation of graphics for the screen, video and web.

Studio Lab I

Studio Lab I
  • Students learn the common threads linked to various multimedia industries.
  • Students learn dSLR photography, file management, project management and presentation practices.
  • Guest speakers and field trips are planned, when possible, to give a real-life view into various related industries.

UX Design

UX Design
  • Students explore the essential design elements and layout components, which are used in creating rich unique user experiences.
  • Students learn the planning and development of a professional level layout for the web, documentation, video, animation or interface which is key to any successful media based project.
  • opics include colour, typography, information hierarchy, common look and feel, psychology and the design brief.

Video Composition

Video Composition
  • Students are introduced to non-linear video editing and composition.
  • Adobe Premiere Pro is used in the importation, assembly and exportation of dynamic video products destined for the web, screen or hand held devices.

Web Development

Web Development
  • Students gain an understanding of the current and next-generation mark-up languages for web development, including HTML, CSS and JavaScript
  • A content management system (CMS) is also studied.

Level 2

Databases & Serverside Development

Databases & Serverside Development
  • Students explore more advanced technologies, such as PHP and MySQL and how they relate to dynamic web development.
  • Students move from client-side HTML and CSS coding to server-side development by working with back-end web-based programming languages.

Motion Graphics Effects

Motion Graphic Effects
  • Students learn how to create motion graphic videos, visual effects for films, and how to combine animations with audio for use in media-based projects
  • The focus of this course is to help the student develop an understanding of the principles of animation and the various concepts of UX in motion graphics.
  • Students will further develop a critical eye in animation to help create visually pleasing and fluid videos through learning different motion graphic techniques.

Studio Lab II

Studio Lab II
  • Students focus on portfolio and resume development, and examine today's media-related industries.

Client-Side Development

Rich Media Services
  • Students are given the opportunity to create dynamic and interactive content for the web using various scripting techniques
  • Technologies covered include topics, such as AJAX, jQuery, JSON, XML and PHP.

Dynamic Media

Dynamic Media
  • Students are introduced to dynamic media and the standards for creating enhanced presentations, animation and interfaces.
  • Possible topics include animation for the web, 3D and other associated tools used to create rich dynamic media.

Client Projects

Client Projects
  • Students are provided a hands-on approach to a live-client project in which they learn the necessary steps in establishing a working relationship with clients.
  • Students plan, manage and develop a real-time multimedia based product, while developing teamwork practices and ethics while working in teams.
